# Korea News Reader

Open `index.html` in a browser to read saved daily Korea news briefs in English.

The daily archive lives in `news-data.js`. New editions should be added to the top of `window.NEWS_ARCHIVE`, while older editions stay below for review.

Each daily edition should include:

- 5 to 7 main Korea-related news stories
- Source name and original article link
- Simple English summary
- 3 useful words or phrases
- 1 thinking question for English practice

Do not copy full article text. Use short summaries and links to the original sources.
